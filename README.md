# project-management-offsite-drone-based-construction-visibility-platform
Designed and managed an end-to-end drone-based construction visibility platform (OFFSITE) integrating LiDAR UAVs, AWS cloud pipelines, and 3D dashboards. Covers full project lifecycle: initiation, planning, execution, risk management, $3.9M budgeting, stakeholder analysis, WBS, and RACI matrix.
