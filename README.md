# 🚁 Autonomous Site Intelligence Platform 

> **Project Management · Engineering · Visual Assets · Product Visualization**

Designed and managed an end-to-end drone-based construction visibility platform (OFFSITE) integrating LiDAR UAVs, AWS cloud pipelines, and real-time 3D dashboards. Covers the full project lifecycle: initiation, planning, execution, risk management, $3.9M budgeting, stakeholder analysis, Work Breakdown Structure (WBS), and RACI matrix.

---

## 🎯 Project Overview

OFFSITE is a digital construction monitoring solution that transforms raw aerial LiDAR data into actionable 3D spatial intelligence. The platform eliminates the schedule delays and cost overruns that affect 98% of megaprojects by providing real-time site visibility, automated cloud processing, and centralized stakeholder communication.
